## การติดตั้ง

First, run the development server:

```bash
npm install
# หรือ
yarn
```

เปิดใช้ [http://localhost:3000](http://localhost:3000) บนเว็บบราวเซอร์เพื่อแสดงผล

สามารถเริ่มพัฒนาได้ตั้งแต่ `pages/index.js`. เป็นต้นไป หรือ `components/`.
